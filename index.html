// The Story

About Me

Hello! My name is Muhamad Fajri, an Informatics Engineering student at Universitas Pasundan, Bandung. I have solid experience in architecting and delivering both web and mobile applications for government institutions, organizational ecosystems, and private clients.

I operate as a Fullstack Developer, specializing in Laravel, Vue.js, Flutter, MySQL, and PostgreSQL. My technical practice encompasses microservices backend design, building centralized API gateways, RESTful API development, and responsive user interfaces.

Driven by code quality and system performance, I focus on building production-ready digital solutions that are stable, highly available, and scalable. I am always open to new professional collaborations and technical challenges.

// The Progression

My Journey

Stage 1: Learning JavaScript

Mastering the Web Core

I began my coding journey by mastering JavaScript fundamentals. I focused on scripting DOM elements, understanding variables, loops, arrays, and standard algorithm design to build interactive features on static webpages.

Stage 2: Frontend Slicing

UI Precision with Vue.js & CSS

I quickly transitioned to building clean client interfaces. I learned how to slice Figma designs into pixel-perfect frontend layouts using Vue.js, styling them with Tailwind CSS and Bootstrap to manage responsive web components.

Stage 3: Deep Backend Study

Server-Side Programming with Laravel

To support complex dashboard actions, I turned my attention to the backend. I mastered Laravel, deep diving into relational databases (MySQL, PostgreSQL), developing unified RESTful API architectures, and learning server routing.

Stage 4: Becoming a Fullstack Developer

Integrating End-to-End Enterprise Systems

By connecting Laravel APIs to Vue.js admin pages, I became a Fullstack Web Developer. I delivered enterprise management systems (Fuel Terminal Cikampek), logistics schedulers, and geographic tracking dashboards for regional government transport offices (Dishub Purwakarta).

Stage 5: Mobile App Engineering (Current)

Cross-Platform Native Apps with Flutter

Currently, I am learning Flutter to design cross-platform native apps. I've designed and synchronized RESTful APIs with microservices architectures, supporting tracking dashboards (Wasil, SIRKEL) on Android and iOS devices.
